# [ElmPrague](https://www.meetup.com/ElmPrague/): Game of Life Dojo

## Requirements

- [elm](http://elm-lang.org/)
- [nvm](https://github.com/creationix/nvm)

Then run:

```sh
nvm install `cat .nvmrc`
nvm use
npm install --global yarn
```

## Install

```sh
nvm use
yarn install
elm package install -y
cd tests
elm package install -y
cd ..
```
